Test Instructions

  1. Click the extension icon in the toolbar
  2. Click "Enable on this tab"
  3. Click "Generate Tone" then play the audio
  4. Adjust the volume slider and verify audio changes
  5. Test preset buttons (0%, 100%, 200%, 400%, 600%)
  6. Click "Add Dynamic Media" to test MutationObserver
  7. Test with multiple tabs for isolation

Tone Generator

Generate a same-origin audio file for testing. This creates a 440Hz tone saved as a local blob.

Static Media Elements

Note: Cross-origin media (YouTube, w3schools, etc.) cannot be boosted by this extension in v1. The browser's CORS policy prevents createMediaElementSource() from processing cross-origin media silently. Use same-origin or blob URLs for testing.
