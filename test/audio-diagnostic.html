What This Checks

  • Shadow DOM audio/video elements (most likely culprit for SoundCloud/Bandcamp)
  • Cross-origin iframes containing media
  • Page's own Web Audio API usage (custom AudioContext)
  • Media Source Extensions (MSE) usage
  • Whether createMediaElementSource() would fail on the media
  • CORS attributes on media elements
